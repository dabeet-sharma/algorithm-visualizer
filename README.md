# 🧠 Algorithm Visualizer

An interactive web-based platform for visualizing classic computer science algorithms in real-time.

Built to deepen my understanding of algorithmic behavior, time complexity, and data structures through visual intuition.

---

## 🚀 Features (Planned)

### Sorting Algorithms
- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort

### Pathfinding Algorithms
- Breadth-First Search (BFS)
- Depth-First Search (DFS)
- Dijkstra's Algorithm
- A* Search

### Data Structures
- Binary Search Trees
- Heaps
- Graph Traversals
- Linked Lists

---

## 🎮 Controls
- Play / Pause
- Step-by-step execution
- Speed control slider
- Randomize input
- Adjustable input size
